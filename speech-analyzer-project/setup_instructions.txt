# Setup Instructions for Speech Analyzer Project

## 1. Create and Activate a Virtual Environment

### On Linux / Mac:
python3 -m venv my_env
source my_env/bin/activate

### On Windows:
python -m venv my_env
my_env\Scripts\activate

---

## 2. Install Required Packages
pip install -r requirements.txt

---

## 3. Run the Applications

### a. Test simple LLM:
python simple_llm.py

### b. Test Speech-to-Text:
python simple_speech2text.py

### c. Run Speech-to-Text Web App:
python speech2text_app.py

### d. Run Full Speech Analyzer (Speech â†’ Summary):
python speech_analyzer.py
